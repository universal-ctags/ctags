:Manual group: Universal Ctags

SYNOPSIS
--------
|	**ctags** ... --languages=+C++ ...
|	**ctags** ... --language-force=C++ ...
|	**ctags** ... --map-C++=+.c++  ...

|	**ctags** ... --map-C++=+.h  ...
